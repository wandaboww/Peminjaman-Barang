```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ                   AUTHENTICATION FLOW                           в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

1. INITIAL STATE (No Login)
   в”ңв”Җ AuthManager::isLoggedIn() = FALSE
   в”ңв”Җ $_SESSION['admin_logged_in'] not set
   в””в”Җ Navbar shows: [Public Mode] [Login Admin]

2. USER CLICKS "LOGIN ADMIN"
   в””в”Җ Redirect to: ?view=login
      в””в”Җ Display login form with password input

3. USER ENTERS PASSWORD (admin123)
   в””в”Җ JavaScript calls: fetch('?action=login')
      в””в”Җ POST request with {password: "admin123"}

4. SERVER VALIDATES PASSWORD
   в”ңв”Җ AuthManager::login($password)
   в”ңв”Җ if ($password === 'admin123')
   в”Ӯ  в”ңв”Җ Set: $_SESSION['admin_logged_in'] = true
   в”Ӯ  в”ңв”Җ Set: $_SESSION['login_time'] = time()
   в”Ӯ  в”ңв”Җ Call: session_write_close() [flush immediately]
   в”Ӯ  в””в”Җ Return: {success: true}
   в””в”Җ else
      в””в”Җ Return: {success: false, message: "Password salah"}

5. JAVASCRIPT RECEIVES RESPONSE
   в”ңв”Җ If success
   в”Ӯ  в”ңв”Җ Show alert: "Login berhasil!"
   в”Ӯ  в”ңв”Җ Wait 500ms [ensure session saved]
   в”Ӯ  в””в”Җ Redirect to: ?view=dashboard
   в””в”Җ If failed
      в””в”Җ Show error message

6. LOAD DASHBOARD
   в”ңв”Җ Server checks: AuthManager::isLoggedIn()
   в”ңв”Җ if TRUE
   в”Ӯ  в”ңв”Җ Load admin version of dashboard
   в”Ӯ  в”ңв”Җ Show 5 admin menus
   в”Ӯ  в””в”Җ Show "Logout" button
   в””в”Җ if FALSE
      в””в”Җ Load public version of dashboard

7. PROTECTED PAGES
   в”ңв”Җ User tries to access: ?view=assets
   в”ңв”Җ Server checks: if (!AuthManager::isLoggedIn())
   в”ңв”Җ if NOT logged in
   в”Ӯ  в””в”Җ Redirect to: ?view=dashboard
   в””в”Җ if logged in
      в””в”Җ Load admin assets page

8. LOGOUT
   в”ңв”Җ User clicks "Logout"
   в”ңв”Җ JavaScript calls: fetch('?action=logout')
   в”ңв”Җ Server executes: session_destroy()
   в”ңв”Җ All session data cleared
   в”ңв”Җ Return: {success: true}
   в”ңв”Җ Redirect to: ?view=dashboard
   в””в”Җ Back to public mode
      в”ңв”Җ AuthManager::isLoggedIn() = FALSE
      в””в”Җ Navbar shows: [Public Mode] [Login Admin]
```
