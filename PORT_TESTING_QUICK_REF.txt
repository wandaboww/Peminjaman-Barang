TROUBLESHOOTING:

Port in use?
  → netstat -ano | findstr :8000
  → Change PORT in script or kill process

PHP not found?
  → php -v (check installation)
  → Add PHP to PATH

Laravel won't start?
  → composer install
  → php artisan key:generate
  → php artisan migrate
