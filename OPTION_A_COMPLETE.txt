### 🔐 **Security Comparison**

#### **BEFORE (❌ Tidak Aman)**
```php
// prototype.php
const ADMIN_PASSWORD = 'admin123'; // ← Terlihat di source code!
```
**Masalah:**
- Password hardcoded di code
- Siapa baca code bisa tahu password
- Tidak bisa change tanpa edit code

#### **AFTER (✅ Aman)**
```php
// prototype.php
private static function getAdminPassword() {
    return $_ENV['ADMIN_PASSWORD'] ?? getenv('ADMIN_PASSWORD') ?? 'admin123';
}
```
**Keuntungan:**
- Password di-load dari `.env` (file terpisah)
- Tidak terlihat di source code
- Bisa change tanpa edit code
- Setiap environment bisa punya password berbeda
